Este é um breve repositório de atividade avaliativa da disciplina de Estrutura de Dados ministrada pelo professor Romulo Menezes. Nas questões abaixo, utilizo os conhecimentos adquiridos ao longo do curso para resolver os problemas propostos com lógica de programação, manipulação do objeto Document e implementação de estruturas de dados. Conhecimento essencial para vida profissional como desenvolvedor.

1ª Questão

Uma demonstração das operações fundamentais de Pilha (LIFO) e Fila (FIFO). O exercício usa os métodos push/pop e shift/unshift para adicionar e remover elementos de um vetor.

2ª Questão

Entendendo o que acontece "por baixo dos panos". A atividade demonstra a lógica de programação pura ao recriar as quatro operações de Pilha e Fila sem a ajuda dos métodos prontos do JavaScript.

3ª Questão

Um mergulho nos métodos de "alta ordem" (Higher-Order Functions) para vetores. A atividade demonstra como usar map, filter, reduce, find e sort para transformar e analisar dados de forma funcional.

4ª Questão

Desconstruindo os métodos map, filter, find, reduce. A atividade foca em recriar a lógica por trás dessas operações, utilizando for e forEach para construir os algoritmos do zero.

5ª Questão

Algoritmos de busca. A atividade implementa a Pesquisa Linear e a Pesquisa Binária, medindo as iterações de cada uma.

6ª Questão

Exercício sobre a dinâmica do DOM (Document Object Model). O script injeta um formulário no HTML, captura os dados do usuário e, em seguida, substitui o próprio formulário por uma mensagem de boas-vindas.

7ª Questão

Exercício de DOM e lógica: uma calculadora que não quebra. O código usa preventDefault para processar um formulário, realizar as 4 operações básicas e, mais importante, validar o input para evitar divisão por zero.

8ª Questão

Um exercício de geometria, constantes e validação. O script aplica as fórmulas de área e perímetro usando Math.PI e, crucialmente, trata a exceção de um raio inválido.

9ª Questão

Um conversor de unidades simples de Celsius para Fahrenheit. O exercício combina a captura de eventos do DOM com a aplicação de uma fórmula matemática para exibir o resultado formatado na tela.

10ª Questão

Indo além da matemática: foco na formatação de saída. O script calcula um salário anual e demonstra o uso da API toLocaleString para formatar o resultado corretamente como moeda brasileira.

11ª Questão

Exercício clássico de lógica condicional: Par ou Ímpar? O script aplica o operador Módulo (%) para verificar o resto da divisão por 2 e apresentar o veredito.

12ª Questão

Uma Tabela Verdade interativa. Este exercício demonstra os pilares da lógica booleana, permitindo ao usuário testar as entradas A e B para ver os resultados dos operadores E, OU e NEGAÇÃO.

13ª Questão

Demonstrando a igualdade estrita (===) em strings. O código verifica se duas palavras ou frases são exatamente iguais, um conceito básico para validação de formulários.

14ª Questão

Uma demonstração dos operadores relacionais (>, <, ===). O script captura dois números e aplica uma cadeia de if... else if... else para determinar se o primeiro é maior, menor ou igual ao segundo.

15ª Questão

Lógica de classificação binária. O script captura um número (idade) e o utiliza em uma estrutura if... else para classificá-lo em uma de duas categorias: maior ou menor de idade.

16ª Questão

Um exercício de aplicação da biblioteca Math. O script captura um número e demonstra o uso direto do método Math.sqrt para encontrar sua raiz quadrada.

17ª Questão

Um exercício de geração dinâmica de HTML. O script captura um número e utiliza um for loop para concatenar uma string HTML, criando uma lista (ul) com os 10 resultados da tabuada.

18ª Questão

Um exercício de lógica composta. O script usa uma cadeia if... else if e o operador && (E) para comparar três números e determinar o maior, tratando também os casos de empate.

19ª Questão

Um exercício de "limpeza" e processamento de strings. O código demonstra o encadeamento de métodos, usando trim() para limpar, split(/\s+/) para dividir, e filter() para contar com precisão as palavras em um texto.

20ª Questão

Um exercício de lógica matemática e otimização. O script implementa um algoritmo de primalidade que checa divisores de forma eficiente, testando apenas até a raiz quadrada (Math.sqrt) do número.

Projetos com Pilha

String Balanceada

Esta é uma página web que verifica o balanceamento de uma string em relação à abertura e fechamento de parênteses, colchetes e chaves. A implementação da lógica central para a validação eficiente dessa estrutura de dados é feita através da utilização da estrutura de dados Pilha.

Navegador

Este projeto reproduz a lógica de navegação (botões Voltar e Avançar) de um navegador web. Ele integra a funcionalidade de busca do Google Search e utiliza a estrutura de dados Pilha para gerenciar e implementar o histórico de páginas, garantindo o funcionamento correto dos botões de navegação.

Hospital
